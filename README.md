# Books_Manager
本科毕业论文项目，基于PHP+MySQL的图书管理系统设计与实现

实现图书系统的基本功能：查询，添加，新增，删除，流通管理，读者信息管理，系统管理，书评，图书热门排行榜，图书新闻，自动化程序等。  <br>
项目访问地址：https://lib.crayon.vip
